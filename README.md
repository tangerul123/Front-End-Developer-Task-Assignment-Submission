# Front-End-Developer-Task-Assignment-Submission

This project is a submission for the Front-End Developer Task Assignment. It focuses on creating a modern, interactive, and visually appealing UI using GSAP animations.

## ✨ Features

- Smooth scroll-triggered animations
- GSAP Timeline & Stagger-based sequencing
- Center card zoom-in focus effect
- Side cards slide-in with rotation
- Fully responsive design

## 🛠️ Technologies Used

- HTML5
- Tailwind CSS
- JavaScript (ES6)
- GSAP (ScrollTrigger)

## 🎯 Animation Highlights

- Section fades in with upward motion on scroll
- Title and subtitle animate from opposite directions
- Main card scales smoothly for focus
- Side cards slide with slight rotation for premium feel
- Hover interactions with smooth scaling
